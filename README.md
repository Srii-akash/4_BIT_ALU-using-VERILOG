# 4_BIT_ALU-using-VERILOG
Designed and implemented a 4-bit Arithmetic Logic Unit (ALU) using Verilog HDL in Xilinx Vivado. The ALU supports arithmetic, logical, and shift operations including addition, subtraction, AND, OR, XOR, NOT, and bit shifting. Verified functionality through simulation, RTL schematics, and testbench-based validation.
